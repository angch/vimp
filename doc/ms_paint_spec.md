# Classic MS Paint Features

## Toolbox (Left Sidebar)
* **Free-Form Select:** Select an irregularly shaped area of the image.
* **Select:** Select a rectangular area of the image.
* **Eraser/Color Eraser:** Erase part of the image; right-clicking replaces the foreground color with the background color.
* **Fill With Color:** Fill an enclosed area with the selected foreground color (Paint Bucket).
* **Pick Color:** Sample a color from the picture to use for drawing (Eyedropper).
* **Magnifier:** Zoom in on a part of the picture.
* **Pencil:** Draw a free-form line one pixel wide.
* **Brush:** Draw using a brush shape (various widths and shapes available).
* **Airbrush:** Draw a spray-can effect.
* **Text:** Insert text into the picture.
* **Line:** Draw a straight line (hold Shift for straight horizontal/vertical/45-degree lines).
* **Curve:** Draw a straight line, then click to pull it into a curve (up to two bends).
* **Rectangle:** Draw a rectangle or square.
* **Polygon:** Draw a shape with multiple straight sides.
* **Ellipse:** Draw an ellipse or circle.
* **Rounded Rectangle:** Draw a rectangle with rounded corners.

## Selection Options
* **Opaque Selection:** The background color in the selection covers the existing image.
* **Transparent Selection:** The background color in the selection becomes transparent, allowing the image behind to show through.

## Image Menu Operations
* **Flip/Rotate:**
    * Flip Horizontal
    * Flip Vertical
    * Rotate by angle (90°, 180°, 270°)
* **Stretch/Skew:**
    * Stretch (percentage horizontal/vertical)
    * Skew (degrees horizontal/vertical)
* **Invert Colors:** Replaces every color with its reverse (negative).
* **Attributes:** Set exact canvas width/height and choose units (Pixels, Inches, Cm).
* **Clear Image:** Erases the entire canvas (Ctrl+Shift+N).
* **Draw Opaque:** Toggle for text box transparency.

## View Options
* **Zoom:** Normal Size, Large Size, Custom (100% - 800%).
* **Show Grid:** Pixel grid (only visible when zoomed in).
* **Show Thumbnail:** A small window showing the full view while zoomed in.
* **View Bitmap:** Displays the image in full-screen mode.

## Colors
* **Color Box:** A palette of 28 default colors.
* **Edit Colors:** Define Custom Colors using the RGB or HSL spectrum.
